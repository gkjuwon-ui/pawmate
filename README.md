# 🐾 PawMate - 반려인 동네 산책 매칭 앱

> **슬로건: "발 맞추는 행복, PawMate"**

## 프로젝트 개요

PawMate는 동네에서 같은 반려동물을 키우는 사람들끼리 산책 매칭을 잡아주는 앱입니다.

## 기술 스택
- **Frontend**: React Native + TypeScript + Expo
- **Backend**: Node.js + Express + TypeScript
- **Database**: PostgreSQL + Prisma ORM
- **Cache**: Redis
- **Real-time**: Socket.IO
- **Maps**: Google Maps API
- **Push**: Firebase Cloud Messaging

## 주요 기능
1. 🐕 반려동물 프로필 등록
2. 📍 동네 기반 산책 매칭
3. 💬 실시간 채팅
4. 📅 산책 일정 관리
5. ⭐ 산책 후기 & 평점
6. 🏥 근처 동물병원/공원 추천
